Java 21 Spring Boot 3.5.5 Spring Cloud 2025.0.0 MySQL 8.x OpenAI GPT API React 19.1.1 Vite 7.1.5 TanStack Query Tailwind CSS Chart.js
주요 기술 선택 배경
  • Spring Cloud (MSA): 서비스별로 독립적 개발/배포가 필요하고, 향후 확장 가능성을 고려하여 MSA 아키텍처 선택. Eureka를 통한 서비스 디스커버리로 유연한 구조 구현
  • MySQL + JPA: 관계형 데이터 모델링이 중요한 프로젝트(상품-가격-리뷰 관계). JPA를 통한 엔티티 중심 개발로 생산성 향상하면서도 복잡한 쿼리는 JPQL과 Fetch Join으로 최적화
  • TanStack Query: 서버 상태 관리와 API 호출 캐싱을 위해 도입. React 19와의 호환성과 강력한 캐싱 기능으로 불필요한 API 호출 감소
  • Chart.js: AI 가격 예측 결과를 시각적으로 보여주기 위해 선택. 라인 차트로 과거 가격 추이와 미래 예측을 직관적으로 표현
