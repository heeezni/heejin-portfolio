Problem - Cause - Solution (Trouble-shooting)

🔴 라이브러리 보안 취약점 대응 (CVE-2025-48924)
Problem 사용 중인 commons-lang3 라이브러리에서 원격 코드 실행 위험 발견
Cause 구버전 라이브러리의 보안 허점으로 인한 비즈니스 리스크 발생
Solution 의존성 버전 정밀 제어(3.18.0)를 통해 취약점 즉시 해결 및 보안 안정성 확보
🔴 분산 환경에서의 인증 보안 취약점
Problem Stateless한 JWT 인증 방식 사용 시 로그아웃 토큰의 무효화 처리 지연
Cause 서버 간 블랙리스트 정보 공유 체계 부재로 탈취된 토큰의 재사용 가능성 상존
Solution Redis 기반 Blacklist 구축으로 로그아웃 토큰 즉시 무효화 및 보안 리스크 원천 차단
🔴 조회 성능 저하 및 서버 부하 급증
Problem 상품 목록 조회 시 응답 지연(3.2s) 발생으로 서비스 이탈 위험 증가
Cause 1:N 관계 엔티티 조회 시 300여 개의 서브 쿼리(N+1 문제)가 발생하여 DB 병목 유발
Solution JPQL Fetch Join 적용으로 쿼리 최적화. 응답 속도 84% 개선 (0.5s)
